# Security Policy

## Supported Versions
We currently support the latest 3 minor versions of the Trinity Symphony ecosystem.

## Reporting a Vulnerability
We welcome security researchers and community members to report potential vulnerabilities. Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them to: **security@dealappseo.com** (or through a private GitHub security advisory).

We will acknowledge your report within 48 hours and provide a detailed response within 7 days.

### Our Promise
We are committed to the safety and ethical integrity of this democratized AI ecosystem. Your contributions help us maintain an unassailable frontier.
